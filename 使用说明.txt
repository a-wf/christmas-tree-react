# 🎄 圣诞树应用 - 使用说明

## 📦 文件说明

### Mac 用户（推荐）：
**圣诞树-1.0.0-arm64.dmg** (约113MB)
- 双击 .dmg 文件
- 拖动"圣诞树"到应用程序文件夹
- 从启动台或应用程序文件夹打开

### Windows 用户：

**方式1 - 便携版（推荐给朋友）：**
**圣诞树 1.0.0.exe** (约154MB)
- ✨ 直接双击就能运行
- 不需要安装
- 可以放在U盘直接运行

**方式2 - 安装版：**
**圣诞树 Setup 1.0.0.exe** (约154MB)
- 双击安装到电脑
- 会在开始菜单创建快捷方式

---

## ⚠️ 首次打开提示

### Mac 用户：
如果提示"无法打开，因为无法验证开发者"：
1. 右键点击应用 → 选择"打开"
2. 在弹出窗口点击"打开"
3. 以后就可以正常双击打开了

**摄像头权限：**
- 首次打开会弹出："圣诞树想要访问摄像头"
- 点击"好"即可
- 如果没弹出：打开"系统设置 → 隐私与安全性 → 摄像头"，勾选"圣诞树"

### Windows 用户：
**安全提示：**
如果提示"Windows已保护你的电脑"：
1. 点击"更多信息"
2. 点击"仍要运行"

**摄像头权限（重要）：**
如果摄像头不工作，需要手动设置：

**Windows 10/11 设置步骤：**
1. 打开"设置" → "隐私和安全性" → "摄像头"
2. 确保"允许应用访问你的摄像头"已开启
3. 在"允许桌面应用访问你的摄像头"列表中，找到"圣诞树.exe"
4. 如果看不到，说明应用可能被系统阻止
5. 解决方法：
   - 关闭应用
   - 右键点击"圣诞树.exe" → "属性" → "兼容性"
   - 取消勾选"以兼容模式运行"
   - 点击"确定"后重新打开应用

**如果还是不行：**
- 检查杀毒软件是否阻止了摄像头访问
- 暂时关闭杀毒软件，允许应用访问后再开启
- 即使摄像头不工作，鼠标控制依然好用！

## 🎮 操作方式

### 首次使用：
1. 允许摄像头权限（可选，用于手势控制）
2. 点击页面以播放背景音乐

### 手势控制：
- ✊ **握拳** = 圣诞树模式（自动旋转）
- ✋ **张开手** = 星系模式（手移到边缘旋转）
- 🤏 **捏手指** = 爱心模式

### 鼠标控制：
- 🖱️ 拖拽 = 旋转
- 🔄 滚轮 = 缩放
- 🖱️ 右键拖拽 = 平移

### 快捷键：
- **H** = 隐藏界面元素
- **ESC** = 退出全屏
